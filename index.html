kegare's Mod

Caveworld　Client / Server

Caveworld ディメンションを追加します。Caveworld は、Y128まで地下層で構成された世界です。
通常世界とは違い、地上が存在しません。最上層と最下層は、岩盤一層で覆われています。

BedrockLayer　Client / Server

岩盤を一層のみの平坦に敷き詰められるようにします。ネザーの岩盤にも対応しています。
生成済みのチャンクにも適用されます。マルチプレイの場合は、サーバ側の導入だけで動作します。

GivingItem　Server

スニークしながら他のプレイヤーを右クリックすることで、手に持ったアイテムを渡すことができます。
マルチプレイ環境向けのMODです。サーバ側の導入だけで動作します。
