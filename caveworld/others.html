その他

Caveworld 2で追加されるその他の要素の説明書きです。

採掘ポイント / Mining Point

CaveworldやCavernで採掘した時に得られるポイントです。
得られるポイントは、コンフィグでブロックごとに設定できます。

確認方法

CaveworldまたはCavernでツルハシを持つと、右上にツルハシアイコンに重なって数字が表示されます。
その数字が現在あなたの保有している採掘ポイントになります。
表示位置はコンフィグで変更できます。

使用用途

@buff コマンド

採掘ポイントを1以上保持していると使用できるコマンドです。
CaveworldまたはCavernで、@buff と発言すると使用できます。
使用時に採掘ポイントを消費し、消費したポイント/秒のバフ(ステータス効果)がプレイヤーに付与されます。
消費量は、@buff [数値] の形で指定できます。入力した数値の分だけ消費します。

炭鉱夫ランク / Miner Rank

保持している採掘ポイントが一定値に達すると、炭鉱夫ランクが上昇します。
炭鉱夫ランクによっていくつかの恩恵を得ることができます。
ランクが上昇する際に、ランク上昇に必要なポイント分だけ減少します。
現在のランクは、採掘ポイントと一緒の位置に表示されます。また、採掘ポイントのツルハシアイコンでも判断できます。
ランクが上昇したときは、チャットログにメッセージが流れます。マルチプレイの場合は、全プレイヤーにメッセージが流れます。
また、CaveworldやCavernにいるプレイヤーには、チャットでの発言時に現在のランクも表示されます。
それぞれの炭鉱夫ランクの表示は、コンフィグから非表示にすることもできます。

初心者 / Beginner

まだ始めたばかりのときのランクです。

石掘り / Stone Miner

採掘ポイントが 50 に達するとこのランクに上昇します。
SextiarySector 2が導入されている場合、このランク以下のとき、CaveworldやCavernで水分が減りやすくなります。

鉄掘り / Iron Miner

採掘ポイントが 100 に達するとこのランクに上昇します。
ここまでに必要な上昇ポイントは、合計で 150 です。

金掘り / Gold Miner

採掘ポイントが 1000 に達するとこのランクに上昇します。
ここまでに必要な上昇ポイントは、合計で 1150 です。

アクアマリン掘り / Aqua Miner

採掘ポイントが 3000 に達するとこのランクに上昇します。
このランク以上になると、水中でも採掘速度が常に遅くならなくなります。
ここまでに必要な上昇ポイントは、合計で 4150 です。

ダイヤモンド掘り / Diamond Miner

採掘ポイントが 10000 に達するとこのランクに上昇します。
このランク以上になると、CaveworldやCavernでの睡眠が、前回の使用時間や使用地点との距離に関係なく行えるようになります。
ここまでに必要な上昇ポイントは、合計で 14150 です。

炭鉱夫 / The Miner

採掘ポイントが 20000 に達するとこのランクに上昇します。
ここまでに必要な上昇ポイントは、合計で 34150 です。

狂った炭鉱夫 / Crazy Miner

採掘ポイントが 50000 に達するとこのランクに上昇します。
ここまでに必要な上昇ポイントは、合計で 84150 です。


コマンド

Caveworldで追加されるコマンドです。

コマンド 説明
/caveworld Caveworld メニューを表示します。メニューからは、バイオームのカスタマイズや鉱脈のカスタマイズ、ディメンションの再生成を呼び出せます。
/caveworld version Caveworldのバージョン情報を、チャットログ(サーバーログ)に出力します。
/caveworld forum CaveworldのフォーラムトピックのURLを開きます。
/caveworld regenerate [true/false] Caveworld ディメンション / Cavern ディメンションの再生成を行います。
クライアントで実行した場合は、Caveworldの再生成GUIを表示します。
regenerateの後の引数には、バックアップの要否を指定します。省略した場合は、バックアップを行います。

レシピ

Caveworldで追加される補助レシピです。
これらはコンフィグで追加の可否を設定できます。

苔石 / 苔石レンガ

苔石は石1個と蔓1個で、苔石レンガは石レンガ1個と蔓1個でクラフトできるようになります。
