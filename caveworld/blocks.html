ブロック

Caveworld 2で追加されるブロックの説明書きです。

ロープ / Rope

ロープは、高所から降りるときなどに便利な、簡易はしごのようなものです。
例えば、峡谷から一気に下層に降りたいときなどに便利です。

使い方

アイテムのロープを手に持って、真上に天井が存在するブロックに使用すると、下へ向かって最大6ブロック分のロープが設置されます。
ロープにカーソルを合わせて使用すると、そのロープからの延長設置をすることができます。
ロープに向かってジャンプしながら接触するか、スニークしながら接触すると、ロープに捕まりやすいかもしれません。
また、地面にまで延長しておくと、すり抜けることなく掴まることができます。


ロープはしご / Rope Ladder

ロープはしごは、ロープをまとめてはしご状に加工したものです。
ロープよりも安全に昇り降りができるようになります。
ロープと同じく、真上に天井のあるブロックに使用すると、下へ向かって最大6ブロック分のロープはしごが設置されます。
スニークしながら真上に天井のある壁に使用すると、壁にロープはしごをくっつけたような状態で設置できます。
はしごと同じように掴まることができ、すり抜けることはありません。
また、ロープはしごの背面でも昇り降りすることができます。

ケイブニウム鉱石 / Cavenium Ore

鉄以上のツルハシで破壊可能で、破壊すると、1～3個のケイブニウムをドロップします。

上質なケイブニウム鉱石 / Refined Cavenium Ore

鉄以上のツルハシで破壊可能で、破壊すると、1～3個の上質なケイブニウムをドロップします。


アクアマリン鉱石 / Aquamarine Ore

鉄以上のツルハシで破壊可能で、破壊すると、1個のアクアマリンをドロップします。


ランダマイト鉱石 / Randomite Ore

石以上のツルハシで破壊可能で、破壊すると、ランダムでいろんな不思議なことが起こります。
アイテムがもらえたり、ステータスエフェクトをもらえたり…
ランダムで得られるアイテムはどれが出やすいというのはありません。全てはあなたの運次第です。


マグナイト鉱石 / Magnite Ore

石以上のツルハシで破壊可能で、破壊すると、マグナイト鉱石をそのままドロップします。

マグナイト鉱石を精錬することで、マグナイトインゴットを得ることができます。


ヘキサイト鉱石 / Hexcite Ore

鉄以上のツルハシで破壊可能で、破壊すると、1個のヘキサイトをドロップします。


インフィタイト鉱石 / Infitite Ore

ダイヤ以上のツルハシで破壊可能で、破壊すると、1個のインフィタイトの欠片をドロップします。
非常に硬い鉱石なので、マグナイトのツルハシを使って採掘するのがおすすめです。


ユニバーサルチェスト / Universal Chest

ワールド(セーブデータ)を越えて中身を共有するインベントリにアクセスできるチェストです。
上質なケイブニウムブロックを8個、エンダーアイを1個使ってクラフトできます。

回収

ユニバーサルチェストは、シルクタッチ付きのツールでのみ回収できます。
それ以外のもので破壊された場合は、そのまま消滅してしまいます。


変質した原木 / Perverted Wood

普通の原木よりも脆く、可燃性の高い原木です。
樫、松、白樺、ジャングルの4種類が存在します。

変質した葉 / Perverted Leaves

普通の葉よりも可燃性の高い葉です。消滅時にたまに変質した苗木をドロップします。
樫、松、白樺、ジャングルの4種類が存在します。

変質した苗木 / Perverted Sapling

普通の苗木よりもはるかに成長しやすい苗木です。普通の苗木とは違い、成長に光を必要としません。
樫、松、白樺、ジャングルの4種類が存在します。

発酵したクモの目と各種類の苗木をクラフトすることで、各種類の変質した苗木をクラフトできます。


アクレシア / Acresia

Cavelandに自生する、成長しやすい作物です。成長に光を必要としません。
また、土以外の石などのブロックの上でも成長します。耕地の上であれば、成長速度がより早くなります。

完全に成長しきると、黄色い実がつきます。

収穫

ブロックを破壊するか、ハサミを持ってブロックを右クリックで、収穫することができます。
収穫すると、アクレシアの実がドロップします。
ハサミを使って収穫するほうがより多く収穫することができ、また成長すれば再収穫することができます。
